AP Exam Strategies

Common patterns, scoring tips, and pitfalls for the AP Calculus exam.

Before You Solve: Read the Problem Right
  • Identify the type first. Spend 5 seconds classifying: separable? linear? logistic? This determines your entire approach.
  • Circle what they're asking for. "Find k" is different from "find y(t)" is different from "find the time when..." — don't solve for more than you need.
  • Check for initial conditions. If they give you \(y(0) = 5\), you'll need it after integrating. Don't lose it in the algebra.
  • Note the units. If the problem says "minutes" but asks for "hours," that conversion is often worth a point.
The #1 Trap: Forgetting the Constant

Every year, thousands of students lose points by forgetting +C or applying the initial condition incorrectly.

  • Always write +C immediately after integrating both sides. Don't "plan to add it later."
  • Apply IC before solving for y. It's often easier to find C while the equation is still in implicit form.
  • If you get \(\ln|y| = \ldots + C\), then \(y = \pm e^C \cdot e^{\ldots}\). Replace \(\pm e^C\) with a single constant \(A\), then use the IC to find \(A\).
  • Don't drop absolute values carelessly. If \(y(0) > 0\) and the solution stays positive, you can drop \(|\;|\). State why.
Separable DEs: The Fast Track
  • Test: can you write it as \(g(y)\,dy = f(x)\,dx\)? If yes, it's separable. If not, try factoring first.
  • Common disguises: \(e^{x-y} = e^x \cdot e^{-y}\) and \(\frac{xy+x}{y^2} = \frac{x(y+1)}{y^2}\). Factor before separating.
  • Don't forget to separate the dy and dx. Write it as two integrals, then integrate each side independently.
  • State the domain. If your solution has \(\ln(x)\), note \(x > 0\). This is often worth a point on free-response.
Slope Fields: What Graders Look For
  • Check where slopes are zero. Set \(dy/dx = 0\) — this gives you horizontal segments. These are the easiest to verify.
  • Check where slopes are undefined. Vertical tangents or asymptotic behavior.
  • "Depends only on x" vs "depends only on y": If \(dy/dx = f(x)\), all segments in a vertical column are parallel. If \(dy/dx = g(y)\), all segments in a horizontal row are parallel.
  • When sketching a solution curve: Start at the IC, follow the flow, and make sure your curve is tangent to the segments — not cutting through them.
Euler's Method: Don't Lose Easy Points (BC)
  • It's just repeated addition: \(y_{n+1} = y_n + h \cdot f(x_n, y_n)\). Set up a table with columns for \(x_n\), \(y_n\), \(f(x_n,y_n)\), and \(h \cdot f\).
  • Show every step. Graders give partial credit for the table even if your final answer is wrong due to arithmetic.
  • Common trap: Using the NEW y-value to compute the next slope (that's the improved Euler method, not what they're asking for).
  • Underestimate or overestimate? If the curve is concave up, Euler's method underestimates. Concave down = overestimates. They love asking this.
Exponential Growth/Decay: Instant Recognition
  • If \(dy/dt = ky\), write \(y = Ce^{kt}\) immediately. Don't re-derive it. The AP exam rewards speed here.
  • Growth: \(k > 0\). Decay: \(k < 0\). Half-life formula: \(t_{1/2} = \frac{\ln 2}{|k|}\).
  • Newton's Cooling: \(dT/dt = k(T - T_{env})\). Solution is \(T(t) = T_{env} + (T_0 - T_{env})e^{kt}\) where \(k < 0\).
  • Two data points → two equations. Use them to find both \(C\) and \(k\). This is the most common free-response setup.
Logistic Growth: The Three Facts (BC)

Almost every logistic question on the BC exam can be answered with these three facts alone:

  1. Carrying capacity \(L\): Factor the DE into the form \(kP(1 - P/L)\). The number multiplying the \((1 - \ldots)\) term gives you \(L\).
  2. Fastest growth at \(P = L/2\): The inflection point of the logistic curve. This is when \(dP/dt\) is maximum.
  3. Solution formula: \(P(t) = \frac{L}{1 + Ae^{-kt}}\) where \(A = \frac{L - P_0}{P_0}\). You rarely need to derive this — just use it.

Trap: If the DE is given as \(dP/dt = 3P - 0.01P^2\), factor it as \(0.01P(300 - P)\) to read off \(L = 300\) and \(k = 3\).

Free-Response Scoring: How to Maximize Points
  • Show separation clearly. Write \(g(y)\,dy = f(x)\,dx\) on its own line. This is typically worth 1 point by itself.
  • Write the antiderivatives. Even if you mess up the algebra later, the integration step earns credit.
  • Include +C and solve for it. These are usually separate points on the rubric.
  • Box or underline your final answer. Make it obvious what you're claiming as the solution.
  • If stuck, write what you know. Setting up the DE correctly is often worth 2-3 points even without solving it.
  • Don't erase work. Cross it out neatly instead. Graders can give credit for crossed-out work if it's correct.
Common Pitfalls (The "I Knew That" Mistakes)
  • Integrating \(1/y\) as \(\ln(y)\) instead of \(\ln|y|\). The absolute value matters unless you can justify \(y > 0\).
  • Forgetting to distribute after exponentiating. \(e^{\ln|y|} = |y|\), not \(y\). Handle the \(\pm\) properly.
  • Mixing up \(dy/dx\) and \(dx/dy\). Double-check which variable you're solving for.
  • Euler's method: using \(\Delta x = 1\) when they said \(\Delta x = 0.5\). Read the step size carefully.
  • Logistic: confusing \(k\) and \(r\). In \(dP/dt = rP(1-P/L)\), the growth rate is \(r\), not \(r/L\).
  • Solving for the general solution when they asked for the particular solution. Always apply the IC if one is given.
Time Management on Exam Day
  • DE free-response: budget 15 minutes. It's usually one of the more straightforward FRQs — don't overthink it.
  • If you're stuck on separation, try multiplying/dividing both sides by something. Factor aggressively.
  • MC questions on DEs: Often testable by plugging the answer back in. If \(y = e^{2x}\) is an option, check if it satisfies the DE.
  • Skip and return. If a DE problem is eating your time, move on. The setup points will still be there when you come back.
